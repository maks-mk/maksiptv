Современный IPTV плеер для любого устройства

MaksIPTV — это мощный плеер для просмотра IPTV каналов с поддержкой плейлистов M3U/M3U8, умной буферизацией и автоматическим переподключением.

Скачать GitHub

Основные возможности

Умное воспроизведение

Автоматическое переподключение при ошибках и интеллектуальная буферизация для стабильного просмотра каналов.

Работа с плейлистами

Поддержка плейлистов M3U/M3U8 с локального диска и из интернета с автоматическим обновлением.

Организация каналов

Удобная навигация по категориям, список избранного, скрытие нежелательных каналов и быстрый поиск.

Комфортный просмотр

Полноэкранный режим, создание скриншотов, настройка громкости и переключение аудиодорожек.

Современный интерфейс

Стильный темный дизайн с минимальной нагрузкой на глаза, адаптивной компоновкой и удобной навигацией.

Кроссплатформенность

Работает на Windows, Linux и MacOS с одинаковым набором функций и без дополнительных настроек.

Установка

  1. Убедитесь, что у вас установлен VLC Media Player
  2. Скачайте последний релиз с GitHub Releases
  3. Распакуйте архив в удобное место
  4. Запустите MaksIPTV.exe

1. Установите необходимые зависимости:

sudo apt install python3 python3-pip vlc python3-pyqt5

2. Клонируйте репозиторий:

git clone https://github.com/maks-mk/maksiptv.git
cd maksiptv

3. Установите Python-зависимости:

pip3 install -r requirements.txt

4. Запустите приложение:

python3 main.py

2. Установите необходимые зависимости:

Использование

Подготовка плейлиста

  1. Используйте встроенный M3U плейлист в программе
  2. Или используйте меню "Файл" → "Открыть плейлист" для выбора любого плейлиста с компьютера
  3. Или используйте меню "Файл" → "Добавить плейлист из URL" для загрузки плейлиста из интернета

Работа с каналами

  • Выбор категории из выпадающего списка
  • Поиск каналов по названию
  • Добавление в избранное через контекстное меню
  • Скрытие нежелательных каналов
  • Сортировка каналов по алфавиту

Управление воспроизведением

  • Воспроизведение/пауза (Пробел)
  • Полноэкранный режим (CTRL+F)
  • Выход из полноэкрана (ESC)
  • Регулировка громкости
  • Создание скриншотов
  • Автоматическое переподключение при ошибках

Управление плейлистами

  • Обновление плейлиста из интернета
  • История недавних плейлистов
  • Проверка доступности каналов
  • Удаление плейлистов из истории

Дополнительные функции

  • Минимизация в системный трей
  • Автоматическое сохранение настроек
  • Логирование работы для диагностики
  • Поддержка горячих клавиш
